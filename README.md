# Javaゲーム開発

## 概要
大学の授業にて、Javaを用いてオセロゲームおよびオリジナルのボードゲーム（ティアラゲーム）を開発しました。

---

## オセロ
### 概要
クライアント・サーバ方式で動作するオセロゲームの中において、AIクライアントを実装しました。
操作を行わずとも、ゲーム開始から終了まで自動で対戦を行うプログラムを開発しています。

### 取り組み
- サーバから送信されるメッセージに応じた処理の実装（ターン管理、盤面更新、ゲーム終了判定など）
- 石を置く座標の決定およびゲーム進行に応じた状態管理の実装

### 工夫した点
- 勝率向上のため、評価関数を設計
  （四隅の取得を優先し、不利な位置である隅周辺を避ける戦略を導入）

---

## ティアラゲーム
### 概要
アニメーションを含むタイルベースゲームをJavaで設計・開発しました。
プレイヤーはキャラクターを操作し、制限時間内でアイテム（ティアラ）を集めるゲームです。

### 取り組み
- ゲームルールの設計
- 盤面管理およびゲーム進行ロジックの実装
- プレイヤー操作に応じた状態遷移の制御

### 工夫した点
- 制限時間を数値とゲージ（バー）で可視化し、直感的に理解できるUIを実現
- アイテム取得数のカウント機能を実装
- ゲームクリア時にキャラクターの見た目が変化する演出を追加し、達成感を向上
